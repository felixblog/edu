Soluzione

Detto U il gruppo di persone della sala, x ∈ U
, ed indicati gli insiemi seguenti:
 
C = { x ∈ U : x porta il cappello }
 
G = { x ∈ U : x porta la giacca }
 
R = { x ∈ U : x porta la cravatta }
 

Il fatto che nessuno porta cappello e giacca si traduce in:

 

C ∩ G = { }

PROBLEMA

ln una sala sono presenti 62 persone, ognuna delle quali ha almeno una delle caratteristiche seguenti: essere europei o essere studenti. Si sa che gli studenti non europei sono 12, che gli inglesi non studenti sono 8, che gli studenti europei non inglesi sono 22 e che gli europei non inglesi né studenti sono tanti quanti gli studenti inglesi. Determina il numero degli inglesi.

 

Soluzione

Detto U l’insieme delle persone della sala, e, considerando che x ∈ U ,

 

S = { x ∈ U : x è studente }
;
 
I = { x ∈ U : x è inglese }
;
 
E = { x ∈ U : x è europeo }
;
 

L’affermazione per cui ogni persona della sala o è europea o è studente si traduce in:

 

U = E ∪ S
 

Inoltre

I ⊂ U
.
 
{ x ∈ U : x è studente non europeo } = S − E
 
{ x ∈ U : x è inglese non studente } = I − S
 
{ x ∈ U : x è studente europeo non inglese } = ( E ∩ S ) − I
 
{ x ∈ U : x è europeo non inglese e non studente } = E − I − S
 
{ x ∈ U : x è studente inglese } = S intersect I
 

La figura seguente riepiloga le informazioni sulla cardinalità degli insiemi (le cardinalità non note sono indicate con “?”):

 

La somma delle cardinalità di tutte le aree è 62, cioè:

 

 

12 + 22 + ? + 8 + ? = 62
 

Semplificando

 

2 ∙ ? = 20
 

Da cui

 

? = 10
 

Per cui gli inglesi sono 10 + 8 = 18.
